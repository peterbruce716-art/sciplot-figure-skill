# Object reconstruction examples

Run `py -3.14 generate_examples.py` from this directory. The script creates three synthetic, privacy-safe examples and their manifests:

- `materials_mechanism`: editable process boxes and semantic connectors.
- `technical_route`: a route diagram with stable object IDs and anchors.
- `micrograph_annotation`: a preserved raster texture plus editable scale bar and annotation.

Each case can be passed to `scripts/object_reconstruction_pipeline.py`. The generated images are synthetic and contain no external paper assets.
