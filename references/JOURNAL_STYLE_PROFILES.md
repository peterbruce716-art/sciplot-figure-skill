# Journal Style Profiles

Profiles in `styles/journal/` are transparent starting points, not current official venue specifications. Each output records its basis, check date, scope, settings, user overrides, and a disclaimer. User settings override a preset, but presets never change data semantics or QA status.

Use the target journal's current author instructions for final dimensions, fonts, line widths, color, and file requirements. The bundled profiles cover generic scientific, Nature-like, Science-like, ACS-like, Elsevier-like, and Chinese-thesis workflows.
